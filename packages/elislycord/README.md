# elislycord
Elisly tarafından kullanılan Discord API ile bağlanmayı sağlayan bir paketidir.

## Özellikler
+ zlib ile sıkıştırmak
+ erlpack (etf) ile şifrelemek
+ 0 cache
+ WebSocket bağlantısı
+ API'ye istek gönderme sistemi
